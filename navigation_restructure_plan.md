## 문제점 분석

### 1. 네비게이션 문제
- 단순 리스트 형태로 모든 시트가 나열됨
- 그룹핑이 없어 찾기 어려움
- 중요도/빈도 구분 없음
- 시각적 계층 구조 부재

### 2. 시트 분류 문제
- 관련 시트들이 흩어져 있음
- 업무 흐름과 맞지 않는 순서
- 카테고리 구분이 불명확

### 3. 가시성 문제
- 사이드바가 단순 텍스트 리스트
- 현재 위치 표시가 약함
- 빠른 접근 경로 부족

### 시각적 개선 사항

1. **그룹 헤더**
   - 배경색으로 구분
   - 접기/펼치기 기능 (선택사항)
   - 아이콘 + 텍스트

2. **현재 위치 강조**
   - 배경색 변경
   - 볼드 처리
   - 아이콘 강조

3. **빠른 접근**
   - 자주 사용하는 시트에 ⭐ 표시
   - 최근 사용한 시트 표시 (선택사항)

4. **색상 코딩**
   - 그룹별 다른 색상
   - 중요도별 색상 차별화

## 구체적 개선 방안

### 1. 사이드바 개선
- 그룹 헤더 추가 (배경색, 볼드)
- 들여쓰기로 하위 항목 표시
- 구분선 추가
- 현재 위치 시각적 강조

### 2. 대시보드 개선
- 그룹별 카드 섹션
- 각 그룹별 빠른 링크
- 시각적 아이콘 강화

### 3. 시트 순서 재배치
- 논리적 흐름에 맞게 재정렬
- 관련 시트들을 가까이 배치

### 4. 색상 체계
- 그룹별 색상 구분
- 중요도별 색상 차별화
