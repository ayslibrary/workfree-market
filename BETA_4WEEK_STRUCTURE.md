## 📊 타임라인 예시

### Week 1 (Day 1-7)
```
신규 모집: ✅
현황: 
- 월요일 가입자 (A팀) → 프로그램 Day 1-7
- 화요일 가입자 (B팀) → 프로그램 Day 1-6
- ...
- 일요일 가입자 (G팀) → 프로그램 Day 1
```

### Week 2 (Day 8-14)
```
신규 모집: ✅
현황:
- A팀 → 프로그램 종료 ✅
- B팀~G팀 → 계속 진행 중
- 새로운 가입자 (H팀) → 프로그램 Day 1-7
```

### Week 3 (Day 15-21)
```
신규 모집: ✅ (마지막 주)
현황:
- H팀 → 프로그램 종료 ✅
- 이전 팀들 계속 종료
- 마지막 배치 가입자 (Z팀) → 프로그램 Day 1-7
```

### Week 4 (Day 22-28)
```
신규 모집: ❌ (마감)
현황:
- Z팀 (마지막 배치) → 프로그램 종료 ✅
- 전체 데이터 수집 완료
- 분석 & 정식 런칭 준비
```

## 💡 왜 이런 구조인가?

### 1. 유연한 가입
```
✅ 3주 동안 언제든 가입 가능
✅ 각자 가입일로부터 7일 진행
✅ 부담 없이 원하는 시기에 참여
```

### 2. 충분한 데이터 수집
```
✅ 100명 전원 7일 프로그램 완수 가능
✅ 4주차에 모든 데이터 수집 완료
✅ 시간대별 분산 참여로 서버 부하 분산
```

### 3. 명확한 마감
```
✅ 3주 후 또는 100명 달성 시 모집 마감
✅ 4주 후 베타 테스트 완전 종료
✅ 정식 런칭 준비 기간 확보
```

## 🎮 개인별 7일 프로그램

각 유저가 경험하는 것:

```
Day 0: 가입 ✅
Day 1: 온보딩 (25분) → 크레딧 30개
Day 2-6: 자유 사용 + 선택 미션
Day 7: 최종 피드백 → 크레딧 10개

총 획득: 최대 크레딧 90개 (9만원 상당)
```
