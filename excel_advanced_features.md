# 엑셀 고급 기능 - 인수인계 문서에 적용 가능한 것들

## 🔥 충격적으로 유용한 엑셀 기능들

### 1. **조건부 서식 (Conditional Formatting) - 시각적 강조**
**왜 충격적인가:**
- 데이터를 자동으로 색상으로 구분
- 규칙을 설정하면 자동으로 적용
- 한눈에 상태 파악 가능

**적용 예시:**
- 완료여부가 "완료"면 초록색, "진행중"이면 노란색, "미완료"면 빨간색
- 오늘 날짜 자동 하이라이트
- 마감 임박 항목 자동 강조 (3일 이내)
- 우선순위별 자동 색상 구분

**실제 활용:**
```excel
=AND(완료여부="미완료", TODAY()-날짜<=3) → 빨간색 배경
=완료여부="완료" → 초록색 배경
=날짜=TODAY() → 노란색 테두리
```

---

### 2. **데이터 유효성 검사 (Data Validation) - 드롭다운 + 규칙**
**왜 충격적인가:**
- 오타 방지
- 일관된 데이터 입력
- 선택지 제한으로 빠른 입력

**적용 예시:**
- 완료여부: 미완료/진행중/완료/보류
- 우선순위: 높음/중/낮음
- 카테고리: 드롭다운으로 선택
- 날짜 범위 제한 (인수인계 기간 내만)

**고급 활용:**
- 다른 셀 값에 따라 드롭다운 목록 변경 (INDIRECT 함수)
- 예: 고객처 선택 → 해당 고객처의 담당자만 표시

---

### 3. **하이퍼링크 + HYPERLINK 함수 - 동적 링크**
**왜 충격적인가:**
- 셀 값에 따라 자동으로 링크 생성
- 다른 시트, 파일, 웹사이트로 연결
- 동적 링크 생성 가능

**적용 예시:**
```excel
=HYPERLINK("#'"&A1&"'!A1", "이동")
=HYPERLINK("file:///C:/Documents/"&B1&".pdf", "매뉴얼 열기")
=HYPERLINK("mailto:"&담당자이메일, "이메일 보내기")
```

**실제 활용:**
- 고객처명 클릭 → 해당 고객처 정보 시트로 이동
- 담당자명 클릭 → 담당자 정보 시트로 이동
- 파일명 클릭 → 실제 파일 열기

---

### 4. **COUNTIF, SUMIF, COUNTIFS - 자동 집계**
**왜 충격적인가:**
- 조건에 맞는 데이터 자동 계산
- 대시보드 진행률 자동 업데이트
- 수동 계산 불필요

**적용 예시:**
```excel
=COUNTIF(완료여부열, "완료") → 완료 항목 수
=COUNTIFS(완료여부열, "미완료", 우선순위열, "높음") → 높은 우선순위 미완료 항목
=SUMIF(카테고리열, "시스템/프로그램", 소요시간열) → 카테고리별 총 소요시간
```

**실제 활용:**
- 대시보드 진행률 자동 계산
- 카테고리별 완료율
- 담당자별 작업량 집계

---

### 5. **VLOOKUP, XLOOKUP - 데이터 연결**
**왜 충격적인가:**
- 다른 시트의 데이터 자동 참조
- 중복 입력 불필요
- 데이터 일관성 유지

**적용 예시:**
```excel
=VLOOKUP(고객처명, 고객사정보시트, 2, FALSE) → 담당자 자동 입력
=XLOOKUP(담당자명, 담당자정보시트, 이메일열) → 이메일 자동 입력
```

**실제 활용:**
- 고객처별 가이드에서 고객사 정보 자동 불러오기
- 담당자명 입력 시 연락처 자동 입력
- SAP t-code 입력 시 사용법 자동 표시

---

### 6. **IF, IFS, SWITCH - 조건부 로직**
**왜 충격적인가:**
- 상황에 따라 다른 값 표시
- 자동 분기 처리
- 복잡한 로직 구현

**적용 예시:**
```excel
=IF(완료여부="완료", "✅", IF(완료여부="진행중", "🔄", "⏳"))
=IFS(우선순위="높음", "🔴", 우선순위="중", "🟡", 우선순위="낮음", "🟢")
=SWITCH(카테고리, "시스템/프로그램", "💻", "업무 프로세스", "📋", "기타")
```

**실제 활용:**
- 완료 상태에 따라 아이콘 자동 표시
- 우선순위별 색상 코드 자동 생성
- 카테고리별 아이콘 자동 표시

---

### 7. **TODAY, NOW - 자동 날짜/시간**
**왜 충격적인가:**
- 항상 최신 날짜 표시
- 수동 업데이트 불필요
- 시간 기반 조건 자동 적용

**적용 예시:**
```excel
=TODAY() → 오늘 날짜 자동 표시
=날짜=TODAY() → 오늘 일정 자동 강조
=DATEDIF(시작일, TODAY(), "d") → 경과 일수 자동 계산
```

**실제 활용:**
- 대시보드에 오늘 날짜 자동 표시
- 오늘 일정 자동 필터링
- 마감까지 남은 일수 자동 계산

## 💡 실제 구현 예시

### 대시보드 자동 진행률 계산
```excel
완료: =COUNTIF(할일목록!G:G, "완료")
진행중: =COUNTIF(할일목록!G:G, "진행중")
미완료: =COUNTIF(할일목록!G:G, "미완료")
총계: =SUM(완료, 진행중, 미완료)
완료율: =완료/총계*100
```

### 오늘 일정 자동 하이라이트
```excel
조건부 서식 규칙:
=TODAY()=인수인계스케줄!A2
→ 노란색 배경 적용
```

### 고객처별 담당자 자동 불러오기
```excel
=VLOOKUP(고객처명, 고객사거래처!A:B, 2, FALSE)
```

### 마감 임박 자동 경고
```excel
조건부 서식:
=AND(완료여부="미완료", 날짜-TODAY()<=3)
→ 빨간색 배경 + 굵은 글씨
```
