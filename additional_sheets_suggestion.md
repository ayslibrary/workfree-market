## 추가 제안 시트

### 1. 📖 업무 프로세스 가이드
**목적**: 각 업무별 상세 프로세스 및 절차 설명

**컬럼 구조**:
- 업무명
- 카테고리
- 단계별 절차 (1단계, 2단계, 3단계...)
- 주의사항
- 관련 시스템/도구
- 참고 문서
- 담당자

**예시 내용**:
- 발주 프로세스
- 출하 프로세스
- 견적 프로세스
- 샘플 처리 프로세스
- FCST 처리 프로세스

### 2. 💻 SAP 사용법
**목적**: SAP 시스템 사용 방법 및 주요 기능 가이드

**컬럼 구조**:
- 기능명
- 메뉴 경로
- 사용 목적
- 단계별 사용법
- 스크린샷 위치
- 주의사항
- 관련 트랜잭션 코드
- 빈도 (자주/가끔/드묾)

**예시 내용**:
- 발주 입력 방법
- 출하 처리 방법
- 재고 조회 방법
- 고객 정보 조회
- 거래처 정보 관리
- 리포트 생성 방법

### 3. 🛠️ 시스템/도구 사용법
**목적**: STARNET 등 각종 시스템 및 도구 사용 가이드

**컬럼 구조**:
- 시스템/도구명
- 용도
- 접근 방법
- 주요 기능
- 사용 방법
- 단축키/팁
- 문제 해결
- 연락처

**예시 내용**:
- STARNET 사용법
- Outlook 설정
- Excel 템플릿 사용법
- 내부 시스템 접근
- 파일 서버 사용법

### 4. ❓ FAQ/Q&A
**목적**: 자주 묻는 질문과 답변 정리

**컬럼 구조**:
- 질문
- 답변
- 카테고리
- 관련 시트/문서
- 업데이트일
- 중요도

**예시 내용**:
- "발주는 어떻게 하나요?"
- "SAP에서 오류가 나면 어떻게 하나요?"
- "샘플 발주는 어디서 하나요?"
- "재택근무 시 어떻게 접근하나요?"

### 5. 📚 용어집/약어 사전
**목적**: 업무에서 사용하는 전문 용어 및 약어 정리

**컬럼 구조**:
- 용어/약어
- 풀네임 (Full Name)
- 의미/설명
- 사용 예시
- 관련 업무
- 카테고리

**예시 내용**:
- APRO, DCC (침퓨즈 관련)
- FCST (Forecast)
- ACCSS (Access)
- EMS (Electronic Manufacturing Services)
- OJT (On-the-Job Training)

### 6. 🔐 시스템 접근 정보
**목적**: 각 시스템별 계정 정보 및 접근 방법 (보안 주의)

**컬럼 구조**:
- 시스템명
- URL/접근 경로
- 계정 정보 위치 (암호화된 위치 참조)
- 접근 권한
- 사용 목적
- 비고

**주의**: 실제 계정 정보는 별도 보안 문서에 저장하고 여기서는 위치만 참조

### 7. 📁 파일 구조/저장 위치
**목적**: 중요 파일들의 저장 위치 및 폴더 구조

**컬럼 구조**:
- 파일/폴더명
- 경로
- 용도
- 업데이트 주기
- 담당자
- 백업 여부

**예시 내용**:
- 매뉴얼 파일 위치
- 템플릿 파일 위치
- 보고서 저장 위치
- 공유 폴더 구조

### 8. ⚡ 단축키/팁 모음
**목적**: 업무 효율을 높이는 단축키 및 팁

**컬럼 구조**:
- 프로그램/시스템
- 기능
- 단축키/방법
- 설명
- 카테고리

**예시 내용**:
- Excel 단축키
- SAP 단축키
- Outlook 단축키
- Windows 단축키
- 유용한 팁

### 9. 🚨 에러 해결 가이드
**목적**: 자주 발생하는 에러 및 해결 방법

**컬럼 구조**:
- 에러 메시지/증상
- 발생 시스템
- 원인
- 해결 방법
- 예방 방법
- 관련 문서
- 연락처

### 10. 📋 업무 템플릿 모음
**목적**: 자주 사용하는 문서 템플릿 및 양식

**컬럼 구조**:
- 템플릿명
- 용도
- 파일 위치
- 사용 방법
- 예시
- 업데이트일

## 우선순위 추천

### Phase 1 (즉시 추가)
1. **업무 프로세스 가이드** - 가장 중요
2. **SAP 사용법** - 사용자가 언급한 핵심
3. **FAQ/Q&A** - 실무에서 바로 활용

### Phase 2 (추가 권장)
4. **시스템/도구 사용법** - STARNET 등
5. **용어집/약어 사전** - 업무 이해도 향상
6. **파일 구조/저장 위치** - 정보 찾기 용이

### Phase 3 (선택 사항)
7. **단축키/팁 모음** - 효율성 향상
8. **에러 해결 가이드** - 문제 해결
9. **업무 템플릿 모음** - 작업 표준화
10. **시스템 접근 정보** - 보안 고려 필요

## 구현 시 고려사항

1. **SAP 사용법 시트**:
   - 스크린샷 첨부 가능하도록 "위치" 컬럼 추가
   - 트랜잭션 코드 별도 컬럼
   - 단계별 상세 설명

2. **업무 프로세스 가이드**:
   - 플로우차트 이미지 참조 가능
   - 각 단계별 체크리스트 연결
   - 예외 상황 처리 방법

3. **FAQ/Q&A**:
   - 검색 기능 강화
   - 카테고리별 필터링
   - 최신순 정렬

4. **용어집**:
   - 알파벳순 정렬 옵션
   - 카테고리별 그룹핑
   - 검색 기능
