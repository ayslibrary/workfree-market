## 1. 사업 개요

### 1.1 비즈니스 모델 정의

**WorkFree는 직장인을 위한 웹 기반 AI 자동화 SaaS 플랫폼입니다.**

| 구분 | 내용 |
|------|------|
| **제품** | 7가지 AI 도구 (블로그, 보고서, 이미지, QR, 이메일, 환율, 챗봇) |
| **타겟** | 마케팅·기획·영업 직장인 (25-40세) |
| **가격** | 크레딧 건당 결제 (1C = 1,000원) + 월 구독 (예정) |
| **플랫폼** | 웹 브라우저 (모바일 최적화) |
| **차별점** | 한글 완벽 지원 + 올인원 + 노코드 + 사용한 만큼만 결제 |

### 1.2 핵심 문제 (Pain Point)

**직장인들의 3대 업무 고통:**

1. **반복 업무 지옥**
   - 매일 블로그 작성, 보고서 정리, 이미지 검색에 2-3시간 소모
   - 60%의 직장인이 "반복 업무로 업무 시간 낭비" 경험

2. **기존 자동화 도구의 한계**
   - Zapier: 복잡한 설정, 학습 곡선 높음
   - ChatGPT: 한국 업무 문화 이해 부족, 프롬프트 작성 필요
   - Notion AI: 월 $10 고정비, Notion 사용자만 가능

3. **비용 부담**
   - 여러 도구 구독 시 월 5-10만원 지출
   - 사용하지 않아도 구독료 발생

### 1.3 WorkFree 솔루션

| 문제 | WorkFree 해결책 | 효과 |
|------|----------------|------|
| 복잡한 설정 | **웹에서 즉시 사용** | 5분 온보딩 |
| 한글 부족 | **한국 업무 맞춤** (기획서, 품의서 등) | 품질 2배 향상 |
| 비싼 고정비 | **건당 결제** (1,000원~) | 월 평균 3만원 |
| 여러 도구 구독 | **올인원 플랫폼** | 7개 도구 하나로 |

**핵심 가치 제안:**
> "하루 2-3시간 절약 = 월 40시간 = 5일치 업무 단축"

## 2. 제품/서비스

### 2.1 현재 제공 AI 도구 (7개)

| 도구 | 기능 | 시간 절약 | 가격 | 구현 상태 |
|------|------|-----------|------|----------|
| 📝 **블로그 생성기** | SEO 최적화 블로그 자동 작성 | 1시간 → 10분 | 3C | ✅ |
| 📊 **보고서 생성기** | 최신 뉴스 기반 산업 보고서 | 2시간 → 15분 | 5C | ✅ |
| 🖼️ **이미지 검색** | 3개 API 통합 (300+ 이미지) | 30분 → 3분 | 1C | ✅ |
| 📱 **QR 생성기** | 대량 QR 생성 + 로고 삽입 | 1시간 → 5분 | 2C | ✅ |
| 📧 **이메일 템플릿** | 비즈니스 이메일 자동 작성 | 30분 → 5분 | 1C | ✅ |
| 💱 **환율 알리미** | 매일 자동 환율 정보 이메일 | 매일 10분 | 2C | ✅ |
| 🤖 **FRI 매뉴얼봇** | AI 챗봇 (RAG 검색) | 20분 → 2분 | 1C | ✅ |

**추가 예정 도구 (6개월 내):**
- 🎨 **AI 초상화 생성기** (2C)
- 📅 **회의록 자동 정리** (3C)
- 📈 **엑셀 차트 생성** (2C)
- 🔍 **경쟁사 분석 리포트** (5C)
- 📞 **콜드메일 자동화** (1C)

### 2.2 핵심 기능

#### 크레딧 시스템
- **1 크레딧 = 1,000원**
- 도구별 차등 가격 (1C~5C)
- 사용하지 않으면 소멸 없음 (영구 보관)

#### 게이미피케이션
- **레벨 시스템**: Bronze → Silver → Gold → Platinum
- **시간 은행**: 절약한 시간을 크레딧으로 전환
- **미션**: 일일/주간 퀘스트 (보상: 크레딧)
- **추천 프로그램**: 친구 초대 시 양측 2C 적립

#### 커뮤니티
- 사용자 간 팁 공유 게시판
- 리뷰 시스템
- 베타 프로그램 (피드백 → 크레딧 보상)

### 2.3 기술 스택 (검증 완료)

```
Frontend: Next.js 15 + React 19 + TypeScript
Backend: Supabase (PostgreSQL) + Firebase
AI: OpenAI GPT-4 + RAG (벡터 검색)
결제: Toss Payments
인프라: Vercel + AWS S3
```

**기술적 완성도:**
- ✅ MVP 완성 (7개 도구 작동)
- ✅ 크레딧 시스템 구현
- ✅ 결제 시스템 통합
- ✅ Analytics 시스템
- 🔄 베타 테스터 모집 중 (목표 100명)

## 3. 시장 분석

### 3.1 시장 규모 (보수적 추정)

#### TAM (Total Addressable Market)
**국내 AI 도구 유료 사용 가능층**
- 대상: 디지털 리터러시 높은 사무직 근로자
- 규모: **약 300만 명**
- 시장 규모: **1.8조 원**
  - (300만 명 × 월 5만원 × 12개월)

#### SAM (Serviceable Addressable Market)
**웹 기반 SaaS 월 결제 의향자**
- 대상: 25-40세 마케팅·기획·영업 직장인
- 규모: **약 60만 명** (TAM의 20%)
- 시장 규모: **2,000억 원**
  - (60만 명 × 월 3만원 × 12개월)

#### SOM (Serviceable Obtainable Market)
**초기 3년 내 목표 시장**
- 1차년도: **5,000명** (SAM의 0.8%)
- 2차년도: **15,000명** (SAM의 2.5%)
- 3차년도: **40,000명** (SAM의 6.7%)

**3년차 목표 매출: 50억원** (40,000명 × 월 35,000원 × 12개월)

### 3.2 시장 트렌드

#### 긍정적 요인
✅ **AI 대중화**: ChatGPT 이후 AI 도구 사용자 300% 증가 (2023년 대비)  
✅ **MZ세대 워라밸 중시**: 생산성 도구 적극 활용  
✅ **노코드 트렌드**: 비개발자도 사용 가능한 도구 선호  
✅ **원격근무 확산**: 디지털 도구 의존도 증가  
✅ **정부 지원**: K-Startup, 예비창업패키지 등

#### 시장 성장률
- **글로벌 AI SaaS 시장**: 연평균 30% 성장
- **국내 SaaS 시장**: 연평균 25% 성장 (2025년 3조 원 예상)

## 5. 경쟁 우위

### 5.1 핵심 차별점

#### 1. 한국형 로컬라이제이션
- **업무 문서 템플릿**: 기획서, 품의서, 시말서 등
- **한국 직장 문화 반영**: 상하 관계, 존댓말/반말, 회사 문화
- **한글 품질**: 자연스러운 한글 표현 (외국 서비스 대비 2배 우수)

#### 2. 올인원 플랫폼
- **7개 도구 통합**: 여러 구독 불필요
- **단일 크레딧**: 하나의 크레딧으로 모든 도구 사용
- **일관된 UI/UX**: 학습 곡선 최소화

#### 3. 건당 결제 모델
- **월 구독 부담 없음**: 사용한 만큼만 결제
- **투명한 가격**: 1C = 1,000원 명확
- **무료 체험**: 신규 가입 시 10C 제공

#### 4. 노코드 & 즉시 사용
- **설치 불필요**: 웹 브라우저만 있으면 OK
- **5분 온보딩**: 회원가입 → 크레딧 구매 → 즉시 사용
- **학습 비용 제로**: 튜토리얼 불필요

#### 5. 게이미피케이션
- **재미있는 경험**: 레벨업, 미션, 업적
- **동기부여**: 시간 절약을 크레딧으로 보상
- **커뮤니티**: 사용자 간 경쟁과 협력

### 5.2 진입 장벽 구축 전략

#### 데이터 축적 (6개월 후)
- 한국 직장인 업무 패턴 데이터 수집
